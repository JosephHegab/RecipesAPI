WELCOME TO THE RECIPES BANK API

This API allows users to create, read, update, and delete recipes. To use the API, you will need to make HTTP requests to the server where the API is hosted https://recipesbank.azurewebsites.net/ Here are some examples of how to use the API:
  • To create a new recipe, make a POST request to the "/recipe" endpoint, with the recipe name and recipe details in the request body. The API will respond with a JSON object containing the new recipe.
  • To get a list of all recipes, make a GET request to the "/recipe" endpoint. The API will respond with a JSON array of all the recipes.
  • To delete a recipe by id, make a DELETE request to the "/recipe/:id" endpoint, with the id of the recipe in the URL. The API will respond with a message indicating that the recipe was deleted successfully.
